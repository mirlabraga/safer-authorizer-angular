## Installation

```bash
$ npm install
```

## Running the app

```bash
# build
$ ng build

# watch mode
$ ng watch

# production mode
$ ng serve
```

## Test

```bash
# Unit Tests with Karma
$ ng test

# To execute the end-to-end
$ ng e2e
```
